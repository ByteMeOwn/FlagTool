# Grand Fantasia Flag Calculator

Ferramenta para calcular flags de itens, classes e encantamentos do Grand Fantasia.

## Propósito

Esta calculadora permite determinar valores de flags utilizados no sistema de itens, classes e encantamentos do jogo Grand Fantasia, facilitando o desenvolvimento e modificação de conteúdo do jogo.

## Uso

Execute a ferramenta e utilize as opções disponíveis para calcular as flags desejadas.
