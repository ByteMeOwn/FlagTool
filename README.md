# Grand Fantasia Flag Calculator
================================

Support tool for calculating and inspecting item, class, and enchantment flags from the game Grand Fantasia, aimed at development, modding, and game content analysis.

### Overview

The Flag Calculator is an application focused on decoding and generating flag values used in the game’s internal structures, such as items, classes, and enchantments.  
Its goal is to simplify the work of those who need to understand, create, or adjust the game’s technical configurations without having to manually calculate bit combinations.

### Features

- Flag calculation for items, classes, and enchantments.  
- Structured visualization of bits and their meanings.  
- Support for different flag categories (e.g.: item type, class restrictions, special properties).  
- Assistance in the process of creating and modifying content for servers or tools related to Grand Fantasia.

### Target audience

- Developers of tools for Grand Fantasia.  
- Administrators and creators of private servers.  
- Modders and enthusiasts who want to understand the internal logic of the game’s flags.

### How to use

1. Run the tool in the supported environment.
2. Select the type of flag to work with.
3. Enter the desired parameters.
4. Check the resulting flag value and, if necessary, copy it for use in configuration files.

### Benefits

- Reduction of errors when manually calculating bit combinations.  
- Faster workflow when creating and adjusting game content.  
- Greater clarity about the meaning of each flag and its composition.
